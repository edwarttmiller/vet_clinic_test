# ВетДоктор – Ветклиника: запись на приём

## Стек
- PHP 8+
- MySQL 5.7+
- HTML5 / CSS3 / JS

## Установка

1. Скопировать папку в `C:\wamp64\www\vet_clinic`
2. Создать БД:
   ```
   mysql -u root -p < database.sql
   ```
3. При необходимости отредактировать `config.php`
4. Открыть: `http://localhost/vet_clinic/`

## Функционал

- Регистрация с валидацией (клиент + сервер)
- Авторизация, сессии
- Запись на приём (вид животного из списка, жалоба, дата, оплата)
- Просмотр своих заявок
- Отзыв только для заявок со статусом «Завершена»
- Администратор: смена статуса, фильтрация, пагинация 5/стр, toast-уведомления
- Слайдер: 4 слайда, автосмена 3 сек, кнопки вперёд/назад
- Адаптация под 390×844 px
